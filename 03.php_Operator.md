# 03. PHP 연산자

## 수학

사칙연산, 나머지 연산, 거듭제곱 연산.<BR>
각각 ```+,-,*,/,%,** ```으로 표현한다.

```php
2+2; //-> 4
17 - 3.5;// -> 13.5
10 / 3; // -> 3.333
6 * 7; // -> 42
4 % 3; // -> 1
2 ** 5; // -> 32
```

## 할당

등호를 사용하여 표현하는 연산자로 , ```$count = $count + 1``` 을 ```$count += 1``` 과 같이 단축하여 표현할 수 있고,<br>
또한 변수에 값을 할당할 때도 사용하여 나타낸다. <br>
이 연산은 ```=, +=, -=, /=, *=, **=, &=, |=, ^=, <<=, >>= ```으로 사용할 수 있다.

```PHP
$message = 'hello, ';
$message = 'world';

$message; // -> Hello, world
```

## 비트

```AND(&), OR(|), XOR(^), NOT(~), Shift(<<,>>)```를 사용할 수 있다. <br>
비트 단위 연산이므로 보기좋게 **2진수**로 표현했고, **AND 는 둘다 1, OR은 하나라도 1, XOR 은 서로 다른 경우, NOT은 반전**이다.

```PHP
$bin = 0b0101; // -> 5

$bin | 0b0100; // -> 0b0100
$bin & 0b0100; // -> 0b0100
$bin ^ 0b0100; // -> 0b0001
$bin << 1; // -> 0b1010
$bin >> 1; // -> 0b0010 
~$bin; // -> 0b1010 = -6
```

## 비교

두 개의 값을 비교할 때 사용한다. <br>
```==, ===```는 같음을 , ```!=, !==```는 같지 않냐고 물어보며, 부등호를 사용하여 크기를 비교한다.<br>
``` >, >=, <, <=```를 기본적으로 사용한다.

```PHP
10 == '10'; // -> true
10 === '10'; // -> false
// 10 <> "10"
10 != '10'; // -> false
10 !=== '10'; // -> true
// 'ab' >= 'ac'
'ab' > 'ac'; // -> false
// '6x' <= 20;
'6x' < 20; // -> true
```

## 우주선 

```<=>```와 같이 사용하고, 두 개의 값을 비교하되, **같으면 0, 왼쪽이 오른쪽보다 작으면 -1, 크면 1을 반환**한다.

```PHP
10<=>20; // -> -1
10<=>10; // -> 0
20<=>10; // -> 1
```

## 증감

```++,--```를 사용하여 **값을 증가시키거나 감소시킨다**. 위치에 따라 **전위 연산**이 있고 **후위 연산**이 있으며 <BR>
```--$count``` 를 전위, ```$count--``` 를 후위라고 한다. 이는 함수 등 다른 것과 함께 쓰일 때 차이를 나타내는데, 후위 연산은 값을 
한 박자 늦게 증가/감소시킨다.

```PHP
$count = 0;

$count--; // -> 0
$count;   // -> -1
```

## 논리

두 개의 값에 대해 ```AND(&&),OR(||),XOR(xor),NOT(!)```을 적용하며 ```true,false``` 값을 반환한다.
느낌은 비트 연산과 비슷하나 그 대상이 ```Boolean```, 또는 그 형태로 반환 될 수 있는 것이다. <br>
예를 들어 []||'Hello, world' 코드는 true를 반환한다.

```PHP
true || true; // -> true
true && true; // -> true
true && false; // -> false
true xor false; // -> true (xor : 서로 다른 경우)
!true; // -> false
```

## NULL

해당 값이 ```null```인지 물어보고, 아니라면 그대로 하고, 그렇다면 뒤에 있는 값을 반환합니다.
```PHP
null ? 'Hello, world'; // -> Hello, world
```


## 삼항 연산자

항이 세개이기 때문에 삼항 연산자이며 두 번째 항은 생략할 수도 있다.
```?```이전에는 조건식이 들어오고, 그 뒤에는 조건식이 참일 경우 할 것, : 뒤에는 조건식이 거짓일 때 할 것을 나타낸다.
다만, 조건식과 참일 때의 할 것이 동일하다면 생략할 수도 있습니다.
```PHP
//'Hello, world'?'Hello, world' :'Bye';
// 조건식과 참일 경우의 항이 동일할 경우 생략 가능
'Hello, world'?: 'Bye';
```

## 문자열
```.```를 사용하면 둘 이상의 문자열을 합칠 수 있다.
```PHP
'Hello, '.'World'; // -> Hello, World
```

## 에러

```@```를 사용하면 발생하는 에러를 무시할 수도 있다. 다만 **속도가 느리기 때문에 추천되지 않는다**.
아래의 코드에서는 MESSAGE 라는 상수가 없으나, 사용하고 있어 에러가 발생하는데, 여기서 @를 사용해서 무시했다.
```PHP
function foo2()
{
    MESSAGE;
}

// @를 이용한 에러 무시
@foo2();
```

## 연산자 우선순위

다른 언어와 마찬가지로 연산자의 우선순위가 있으며, 여러개의 연산자가 함께 쓰일 경우 먼저 동작하는 연산자가 있다.<br>
비교 연산자가 논리 연산자보다 우선순위가 높다. 따라서 10 <20, 10> 20을 먼저하고 논리연산을 진행한다.
```PHP
/*
비교연산자: ==,=== 같냐, !=,!== 같지 않냐, >,>=,<,<=

논리연산자: AND(&&), OR(||), XOR(or), NOT(!)
*/

// 비교연산자가 논리연산자보다 우선순위 높다.
true && 10 < 20 == true && 10 > 20; // -> 
```
