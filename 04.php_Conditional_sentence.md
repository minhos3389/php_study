# 04. PHP 조건문 (if, switch)

## if~elseif~else

```if~elseif~else```문은 다른 언어와 크게 다르지 않다.<br>
조건에 맞으면 ```if```, 안 맞으면 ```elseif,else```를 순차적으로 검사하고 처리한다. 알아야 할 것은 조건을 사용할 때,
```false```로 처리되는 것들이며, ```[],'',false,0,null``` 과 같은 것들은 모두 **거짓**으로 판단된다.

```PHP
// [], '', false, 0, null -> false 
if (true) {
    echo 'Hello, world';
}

// -> Bye           if~else 조건문
if (false) {
    echo 'Hello, world';
} else {
    echo 'Bye';
}


// -> Who are you?  if~elseif~else 조건문
if (false) {
    echo 'Hello, world';
} elseif (false) {
    echo 'Who are you?';
} else {
    echo 'Bye';
}
```
> 비교, 논리 연산자의 반환 값은 ```true``` 또는 ``` false```다. 조건문에서 비교, 논리 연산자를 많이 사용한다.

## 변수 할당하기
- 조건을 사용하면서 변수에 값을 할당하는 것을 허용한다.
```PHP
// -> Hello, world   변수에 값을 할당할 경우 참이된다.
if ($message = 'Hello, world') {
    echo $message;
}
```

## switch~case

```switch~case```구문은 ```if```와 치환이 가능하나, 사용할 때의 느낌이 다르다. <br>
이는 여러 개의 값 중에 하나를 선택한다는 관점에서 접근해야 한다.<br>
변수의 값에 따라 행동이 달라지고, ```break```로 블럭을 빠져나갈 수 있다.<br>
다만 상황에 따라 ```case```를 연속적으로 사용하여 여러 개의 값을 처리할 수 있을지도 모른다.<br>
```default```를 사용하여 기본 행동을 정의한다.

```PHP
$context = 1;

// -> Hello, world
switch ($context) {
    case 1:
        echo 'Hello, world';
        break;
    case 2:
        echo 'Who are you?';
        break;
    // 기본행동 정의 
    default:
        echo 'Bye';
}
```
> if 문으로 변경시

```PHP
if ($context == 1) {
    echo 'Hello, world';
} elseif ($context == 2) {
    echo 'Who are you?';
} else {
    echo 'Bye';
}
```
